# Playable Factory Case Study

A JavaScript-based interactive web application for a playable ad.

### Prerequisites

- [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/)

### Installation and Setup

#### 1. **Install Packages**
npm install
#### 2. **Run the Project**
npm start
#### 3. **Build the Project**
npm run build
